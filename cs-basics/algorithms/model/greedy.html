  1. 区间覆盖

给定N个闭区间[ai,bi]以及一个线段区间[s,t] ，请你选择尽量少的区间，将指定线段区间完全覆盖

  • 将闭区间以左端点大小排序
  • 每一次循环筛选左端点小于start的区间，获取最大的右端点，判断是否大于线段区间右端点
    • 若大于，则结束
    • 若小于，令start=最大的右端点，继续循环

import os
import sys

n, m, x = map(int, input().split())
nums = list(map(int, input().split()))
books = [[] for i in range(n)]
for i in range(m):
    b, l, r = map(int, input().split())
    books[b - 1].append([l, r])
sorted_record = []


def calculate(book, index):
    global sorted_record
    if len(book) == 0:
        return
    current_record = 0
    temp = []
    for record in book:
        if current_record == 0:
            current_record = record
            temp.append(record)
        elif record[0] > current_record[1]:
            current_record = record
            temp.append(record)
    sorted_record.append((temp, index))
    for record in temp:
        book.remove(record)
    calculate(book, index)
res=0
current_x=0
for index, book in enumerate(books):
    book.sort(key=lambda x: (x[1],x[0]))
    calculate(book, index)
sorted_record.sort(key=lambda x:-len(x[0]))
for record in sorted_record:
    if nums[record[1]]>0:
        nums[record[1]] -= 1
        res+=len(record[0])
    else:
        current_x+=1
        if current_x<=x:
            res+=len(record[0])
print(res)
